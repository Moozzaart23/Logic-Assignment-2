struct node* BuildParseTree ( char  expression[])

BuildParseTree() method This function is mainly used to build the parse tree in terms of the root. It's main function is to return the root after carrying out the operation on the postfix form of the expression entered. Here if '~' is encountered, we initialise left node with NULL and right node with the character next to it. For dual noded operators, left and right both nodes are initialised.

Call this function as follow:

struct node *root=BuildParseTree(tree);
Note
tree is a character array
Returns
The function returns the root node.
